## <u>Teamate</u> Project Background
- ### 필요성
  - 제지 산업은 스팀을 사용하는 대표적인 에너지 다소비 산업으로, 에너지 절감과 탄소 배출 저감이 시급함.
  - 특히 초지공정과 건조공정은 설비 간 상호작용과 공정 변동성이 커, 안정적인 품질 유지와 에너지 절감의 병행이 어려움.
  - 최근 산업계에서는 이러한 한계 극복을 위해 스마트 팩토리 전환을 가속화하고 있으며, 그중에서도 디지털 트윈이 핵심 기술로 주목받고 있음.
  - 제지 산업 또한 디지털 트윈을 활용한 공정 시뮬레이션과 데이터 기반 최적화가 에너지 효율과 품질 향상에 필수적이라고 판단함. 
- ### 기존 해결책의 문제점
  - 기존 최적화 연구는 이론적 접근에 머물러 현장 적용성과 실시간 대응력이 부족함.  
  - 공정 데이터의 분산 및 관리 미흡으로 인해 경험 의존적 운전이 지속되고, 객관적 분석과 의사결정이 어려움.
  - 이로 인해 이상 상황에 대한 실시간 대응 체계 구축이 어렵고, 생산성, 품질, 엔지 효율 간의 균형적 제어가 제한되는 한계가 존재함.

## Case Study
  - ### Description
    본 시스템은 실제 제지 공정 데이터를 기반으로 품질과 에너지 효율을 동시에 최적화함.  
    Python 기반 예측 모델을 구축하여 품질 점수와 스팀 사용량을 분석하고,  
    다목적 함수 및 강화학습 모델을 통해 **스팀 사용량 약 6~24% 절감**, **생산량 2.8% 향상**을 달성함.  

    모델 결과는 Flask API를 통해 React 대시보드로 전송되어,  
    품질 점수, 속도, 수분, 압력 등의 공정 지표를 실시간으로 시각화함.  
    유사 공정 비교, 센서 중요도 분석 기능을 통해 작업자의 데이터 기반 의사결정을 지원함.  

    또한 Unreal Engine을 활용한 디지털 트윈 환경을 구축하여,  
    사용자가 입력한 운전 조건에 따른 품질 및 에너지 변화를 시뮬레이션함.  
    이를 통해 실제 운전 전 단계에서 최적 운전 전략을 검증하고,  
    에너지 절감과 품질 안정화의 가능성을 확인함.  
  
  
## Conclusion
  - ### 모델링 기반 에너지 최적화  
    다목적 함수와 강화학습 모델을 적용하여 **스팀 사용량 약 6~24% 절감**, **생산량 2.8% 향상**을 달성함.  
    품질과 에너지 효율의 균형을 확보하며 최적 운전 전략의 효과를 입증함.  
  
  - ### 데이터 기반 운전 안정화  
    공정 데이터를 통합 관리하여 품질 편차를 줄이고 안정적인 운전 환경을 구축함.  
    운전자의 경험 의존도를 낮추고 데이터 중심의 의사결정을 가능하게 함.  
  
  - ### 실시간 시각화 및 대응력 향상  
    React 대시보드를 통해 공정 상태를 직관적으로 모니터링함.  
    실시간 데이터 갱신으로 작업 효율과 이상 대응 속도를 향상시킴.  
  
  - ### 디지털 트윈 기반 운전 전략 검증  
    Unreal Engine 기반 가상 공정 환경에서 사용자가 입력한 운전 조건을 시뮬레이션함.  
    입력값에 따른 품질 및 에너지 변화를 검증하여 최적 운전 전략 수립에 활용 가능함.  
  
  - ### 산업 확장성  
    본 시스템은 제지 산업을 넘어 다양한 제조 공정에도 적용 가능함.  
    데이터 분석·시각화·시뮬레이션을 통합한 스마트팩토리형 플랫폼으로 확장 가능함.
